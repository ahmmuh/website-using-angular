React js

React är ett effektivt och flexibelt Javascript bibliotek som används för att bygga användargränsnitt. Enkelt förklarat är ett Javascript bibliotek ett typ av bibliotek som har färdiga funktioner man använder sig utav när man skriver Javascript. Ett bibliotek gör det alltså enklare att skriva komplex kod

React Native

React Native är ett ramverk för apputveckling med framförallt JavaScript. Det skapades av Facebook och har på kort tid blivit en stomme för apputvecklare

Vue js

Vue är ett ramverk som kan användas till att bygga användargränssnitt i webbläsaren. Det är ett alternativ till andra populära ramverk

Django är ett ramverk för utveckling av webbsidor. Django är skrivet i Python. ... Syftet med Django är att göra det enklare att utveckla avancerade databas-drivna webbplatser.
